Difference between var, let and const with code examples.

Var

  • declarations are globally scoped
  • variables can be updated and re-declared within its scope
  • var a=10;

    • Let

  • let is block scoped
  • let b=20;

    • Const

  • const is block scoped
  • cannot be re-assigned to a new value
  • const PI=3.14;
